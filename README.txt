README	FolloweMe.exe - simulation of action behavior in Ableton's engine as descirbed in Task 2

AUTHORS	Kamil Piekutowski

COMPILE	
To compile:
1. Open  FolloweMe.sln in FolloweMe folder.
2. Build or rebuild solution. 
3. Run with local Debugger.

To run from command line:
$cd FolloweMe/Debug
$FolloweMe.exe or $FolloweMe.exe < commands.txt

USAGE
Example:  ./FolloweMe.exe < commands.txt

1. Where redirection: "< commands.txt" is optional and could any file
that contains proper commands.
